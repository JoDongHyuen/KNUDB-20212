!! phase2 insert하는 쿼리문이 조금씩 변경이 생겼습니다.
그래서 기존 데이터는 지우고 새로 insert 해주시면 감사하겠습니다.
변경사항이 생긴 부분은 함께 첨부하였습니다.

사용방법

homepage.jsp 를 통해 run을 합니다.
그러면 메인페이지가 뜨게 되는데

메인페이지에서는 회원가입을 하지 않아도 쿼리3개를 누구나 볼수 있도록 띄워 줍니다.
그리고 하얀색 searching 버튼을 통해서 phase2에서 짰던 쿼리를 통해 검색을 할 수 있도록 활용하였습니다.

회원가입은 customer / owner 두 가지로 나뉘고 회원 가입을 진행 했을 시 공통적으로 information table에 정보가 업데이트 되며
고객의 정보는 Customer 테이블에 들어가게 되고, 점주의 정보는 Owner 테이블에 들어가게 됩니다.
여기서 주목할 부분은 점주 정보쪽에 transaction이 들어가있는데 이부분은 additional txt에 기술하였습니다.

로그인 했을시에는 customer / owner / admin 세가지 기능을 나뉘게 되면서
customer 에서는 개인정보변경, 음식점 예약, 음식점 평점 매기기가 가능합니다
owner 에서는 음식점 이름 변경, 음료수 추가, 음식 추가, 음식 가격 변경. 개인정보 변경이 가능합니다. 왼쪽에 search 버튼을 누르면 phase3 에서 만든 여러 쿼리들을 매개변수를 넣어서 실행 할 수 있습니다. owner 에서도 마찬가지로 phase3 에서 만든 쿼리들을 넣어서 제작했습니다. admin 으로 로그인을 하면 고객과 점주들의 정보를 한눈에 볼 수 있습니다.
admin 에서는 고객, 점주들의 정보를 확인 할 수 있으며 점주 테이블에서는 이름을 클릭할 수 있는데 이름을 클릭하게되면
해당 점주가 소유하고 있는 가게 정보를 동적으로 보여줍니다.
